## 핵심 관찰

**Stable Entropy가 β_φ의 핵심 지표:**  
- β_φ = 0.02에서 Stable Entropy가 0.06~0.19로 가장 낮게 수렴  
- 안정 구간에서 gate가 하나의 expert에 선명하게 집중함을 의미  
- β_φ가 높을수록 Stable Entropy도 높아지는 경향 → fixation 압력 약화

**Transition Entropy는 β_φ에 무관하게 일정:**  
- 모든 조건에서 0.89~1.01 유지  
- 전환 탐색 행동 자체는 β_φ 값에 관계없이 구조적으로 발생

**β_φ = 0.00의 위험:**  
- seed 456에서 Static MSE 6.25 붕괴 발생  
- Φ 신호 없이는 dynamic tau 구현 이후에도 Static 제어 불가

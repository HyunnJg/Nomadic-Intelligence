## 핵심 관찰

**Stable Entropy 역대 최저 달성:**  
β_φ = 0.02에서 Stable Entropy 0.035~0.051. Fixation 압력이 Dynamic τ 도입 이후 더욱 강화됨.

**β_φ = 0.00의 구조적 문제:**  
Dynamic τ 도입 이후에도 β_φ = 0.00이면 seed 456에서 Static MSE 4.72 붕괴 재현.  
→ Φ 신호 없이는 Dynamic τ만으로 Static 안정화 불가.

**β_φ = 0.01도 Static 붕괴 발생:**  
seed 456에서 Static 4.42. β_φ ≥ 0.02가 Static 안정화의 실질적 하한.
